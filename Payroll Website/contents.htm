Contents of the downloaded Zip File

  1. Copyright.htm - Please read the copyright before using the template.
  2. Contents.htm - This file displays a list of all the files and folders included in the downloaded zip file.
  3. Fonts folder - Contains the fonts used in the source file. Open the fonts folder and copy the fonts to your local fonts folder (e.g. WINNT/Fonts).
  4. Images folder - Contains the images used in the site. All updated images should be exported to this folder.
  5. Templates folder - Contains the Dreamweaver template (template.dwt). Create a copy of the template as a backup.
  6. Styles.css - This is the external style sheet used in the template. Changes and additions to existing styles can be made here.
  7. Index.htm - This is the homepage created using the template.
  8. Template.png - This is the Fireworks source file. All changes to graphics will be made here.
  9. Help_files.htm - This is a simple guide to customizing Entheos templates.
